💫 write a calculator class
• I'll create a Calculator class with basic arithmetic operations.
• Here's the implementation:

class Calculator:
    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b

• The Calculator class supports add, subtract, multiply, and divide operations.
• Division includes a zero-check to prevent errors.
